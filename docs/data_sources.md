# Data Sources

This project aims to estimate hospital system pressure in the Ruhr region using official or publicly available datasets.

## Geographic scope

Primary scope:

- Ruhrgebiet / Ruhr region
- City-level analysis for major Ruhr cities
- Later: hospital-level analysis where public data is available

Core cities:

- Essen
- Dortmund
- Duisburg
- Bochum
- Gelsenkirchen
- Oberhausen
- Mülheim an der Ruhr
- Herne
- Hagen
- Bottrop

## Target indicators

### Hospital capacity

- Number of hospitals
- Number of beds
- Bed occupancy rate
- Average length of stay
- Stationary patients
- Treatment days / occupied bed days

Possible sources:

- IT.NRW
- Destatis
- Krankenhausverzeichnis
- Krankenhaus Qualitätsberichte

### Workforce

- Hospital physicians
- Nursing staff / Pflegepersonal
- Full-time equivalents where available

Possible sources:

- IT.NRW
- Destatis
- Bundesärztekammer
- hospital quality reports

### Demographics

- Population
- Population aged 65+
- Population change

Possible sources:

- IT.NRW
- Destatis
- city statistical offices

### Socio-economic indicators

- Unemployment rate
- Bürgergeld / social benefit rate
- Income proxy if available

Possible sources:

- Bundesagentur für Arbeit
- IT.NRW
- city statistical offices

### Regulation / staffing reference

Important references:

- Pflegepersonaluntergrenzen-Verordnung (PpUGV)
- §137i SGB V
- Pflegepersonalbemessungsverordnung / PPR 2.0
- G-BA Notfallstrukturen

## Methodological notes

The first version of the project uses city-level indicators.

Hospital-level analysis will be added only where reliable public data is available.

Department-level analysis is optional and limited to critical areas such as ICU, Stroke Unit, Geriatrics, Internal Medicine, or Emergency Care.

The Hospital Pressure Index is a transparent composite indicator, not a clinical safety verdict.
